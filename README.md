## Getting Started

### Installation
1. Clone the repo
   ```bash
   git clone https://github.com/steelWinds/VVibes
   ```
2. Install NPM packages
   ```bash
   npm install
   ```
3. Build project
   ```bash
   npm build
   ```
3. Run project!
   ```bash
   npm preview
   ```
